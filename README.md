# 🎯 AI-Powered Python Quiz Application

An interactive web-based quiz application built with **Python**, **Flask**, **HTML**, **CSS**, **JavaScript**, and **SQLite**. The application allows users to take quizzes, receive instant feedback, view their scores, and interact with an AI-powered chatbot for assistance.

---

# 📖 About the Project

The AI-Powered Python Quiz Application is designed to provide an engaging learning experience through interactive multiple-choice quizzes. It offers a clean and responsive interface, automatically evaluates answers, stores quiz results, and displays detailed score reports.

This project demonstrates full-stack web development using Flask and serves as a practical example of integrating frontend and backend technologies into a real-world educational application.

# ⚙ Installation

### Clone the Repository

```bash
git clone https://github.com/ramasudha428-code/QUIZ_PROJECT.git
```

### Navigate to the Project Directory

```bash
cd QUIZ_PROJECT
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
